# Inventory Management System

## Project Overview

The Inventory Management System is a full-stack web application developed using React.js, Node.js, Express.js, and MySQL.

The application helps businesses manage products, inventory, orders, and users efficiently.

## Features

* User Authentication
* Role-Based Access Control
* Product Management
* Inventory Tracking
* Order Management
* Stock Updates
* Dashboard Analytics
* Audit Logging

## Technology Stack

Frontend:

* React.js
* Axios
* Bootstrap

Backend:

* Node.js
* Express.js
* JWT Authentication
* bcrypt

Database:

* MySQL

## Installation

### Backend

npm install

npm start

### Frontend

npm install

npm run dev
