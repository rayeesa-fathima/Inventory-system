# Inventory Management System

## Project Overview

The Inventory Management System is a web-based application designed to manage products, inventory, and customer orders efficiently.

## Features

* User Authentication (Login)
* Product Management
* Inventory Tracking
* Order Management
* Dashboard Overview
* MySQL Database Integration

## Technology Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Installation

### Backend Setup

1. Navigate to backend folder

```bash
cd backend
```

2. Install dependencies

```bash
npm install
```

3. Start backend server

```bash
node server.js
```

### Frontend Setup

1. Navigate to frontend folder

```bash
cd frontend
```

2. Install dependencies

```bash
npm install
```

3. Start frontend

```bash
npm run dev
```

## Database

Database Name:

```sql
inventory_db
```
